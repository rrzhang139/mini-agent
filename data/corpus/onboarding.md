# New Employee Onboarding Guide

## Week 1 Checklist

- [ ] Complete IT setup (laptop, credentials, VPN)
- [ ] Attend orientation session
- [ ] Review company policies
- [ ] Set up benefits enrollment
- [ ] Schedule agent walkthrough with Xi

## Agent Walkthrough

All new employees should schedule an agent walkthrough session within their first week. This helps you understand:
- How to interact with our AI agents
- What questions the agents can answer
- How to verify information with citations
- Tool capabilities and limitations

## Relocation Support

If you're relocating:
1. Review the relocation policy document
2. Submit banking information through HR portal
3. Expect payment within 5 business days of submission
4. Keep receipts for tax purposes
